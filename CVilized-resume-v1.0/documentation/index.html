D.) CSS and SASS (Compass) - top

This template is built with SASS, means that the CSS file located in /assets/css/main.css is generated by the compiler instead of manually written. We recommend that you use SASS to customize this template, it will be quick and fast. However you can also customizing the template without SASS by directly editing the file /assets/css/main.css

  1. D1.) Why Customizing With SASS?

    Let's put a simple definition about SASS, it allows us to create programmable stylesheet (CSS). This theme use .scss file to be compiled into .css, so you don't write .css directly. This guide only cover SASS variables used in the theme, such as color and font. Then what is Compass? Let's say it extends the functionality of SASS. For further information, please go to the official website of SASS and Compass.

  2. D2.) SASS File Structure

    You will find all the .scss file under sass directory. It has strucuture as below:

    • main.scss
    • /modules/
      • _mixin.scss
      • _variables.scss
    • /partials/
      • _about-me.scss
      • _base.scss
      • _contacts.scss
      • _footer.scss
      • _header.scss
      • _navigation.scss
      • _projects.scss
      • _resume.scss
    • /vendor/
      • _bootstrap.scss
      • _font-awesome.scss
      • _timeline.scss

    /modules/_variables.scss: hold all variables used in this template, such as base font color, section background color, base font family and many more. You need to change the variables value in this file to customize the template with your chosen color, font, background image etc.

    /partials/: each .scss file (except _base.scss) in this directory provides style for each section of the template such as Navigation, Header, About Me and so on. The file name is self-explanatory, so you can easily find the appropriate file for certain style

    /partials/_base.css: this file is used to define general styles applied to the whole template such as body, link, icon and other general styles.

    /vendor/: includes all the 3rd party files or libraries. You shouldn't change the content of the files or remove the files in this directory. Somehow, you can add another 3rd party files necessary for your customization in this directory.

E.) Javascript - top

You can find all the javascript files in the /assets/js/ directory. Please note that the directory mainly consists of framework or plugin scripts. The only file that specific to the template and we can change is main.js

What does main.js do?

  • Initiate scrollspy. When you scroll the page, the main navigation is pointing to the menu based on your current scrolling position.
  • Initiate smooth page scrolling
  • Initiate google maps. Also, you should go to this initiation code to change the map address.
  • Cross-browser client-side form validation. This handle browser who doesn't have HTML5 validation.
  • Ajax contact form. Initiate and do ajax call to send the input data to the contact.php to be processed in the server side.
